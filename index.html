Freeitem

Module for freeitem (magento)

Download as .zip Download as .tar.gz View on GitHub

Información:

El módulo free item añade un producto simple o configurable al carrito cuando la cantidad del carrito es superior a la indicada.

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.

Requisitos:

  1. modman instalado. -> https://github.com/colinmollenhour/modman
  2. modman init -> en raiz del magento donde queremos instalar el módulo.
  3. modman clone git@github.com:DevopensourceTeam/FreeItem.git

Características:

  • Productos configurables y simples.
  • Añade el producto de regalo cuando la cantidad del carrito es superior a la cantidad indicada en la configuración del módulo en el backend.
  • Elimina del carrito el producto de regalo si la cantidad es inferior a la cantidad indicada en la configuración del módulo en el backend.
  • Cantidad del producto de regalo limitada a 1.
  • Precio 0€ para producto regalo.
  • Al actualizar carrito añade el producto de regalo si la cantidad es superior a la cantidad indicada en la configuración del módulo en el backend.
